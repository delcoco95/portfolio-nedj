Projets Systèmes et Réseaux

Dans le cadre de mes études en BTS SIO option SISR, j’ai réalisé plusieurs projets réseaux mettant en œuvre des environnements Windows et Linux. Ces projets m’ont permis de travailler sur la virtualisation, la gestion des identités, l’automatisation via scripts, ainsi que la mise en place de services réseau sécurisés.
Outils utilisés : VirtualBox, Hyper-V, Windows Server 2022, Debian, PowerShell, Docker, Python3.

Déploiement d'un contrôleur de domaine Active Directory

Mise en place d'un annuaire centralisé avec gestion des utilisateurs et des OUs sur un domaine testsrv.local simulé en environnement virtuel.

⏱️ 1 jour – Technologies : Windows Server 2022, AD DS, DNS, DHCP, VirtualBox
Automatisation du déploiement de services Windows Server 2022 via PowerShell

Développement d'un menu PowerShell interactif automatisant le déploiement complet d'un serveur.

⏱️ 22 jours – Technologies : PowerShell, Windows Server 2022, AD DS, DNS, DHCP, VirtualBox
Serveur de messagerie sécurisé — ClassCord

Déploiement d'un serveur de messagerie Python multi-clients avec authentification SHA-256, gestion de canaux et conteneurisation Docker.

⏱️ 5 jours – Technologies : Python, Docker, SQLite, UFW, Debian, VirtualBox
Supervision réseau — Prometheus, Grafana & Alertmanager

Mise en place d'une chaîne de supervision complète sur VM Debian : collecte de métriques, dashboards Grafana et alertes automatiques par mail via Postfix.

⏱️ 2 jours – Technologies : Prometheus, Grafana, Node Exporter, Alertmanager, Postfix, Debian
Déploiement de GLPI — Gestion de parc informatique

Installation et configuration de GLPI sur VM Debian avec agent d'inventaire automatique pour la gestion centralisée du parc.

⏱️ 5 jours – Technologies : GLPI, Debian, Apache2, MariaDB, PHP, VirtualBox
Sécurisation serveur Linux multi-services — Mediaschool

Déploiement automatisé via Vagrant d'un serveur Ubuntu sécurisé hébergeant OpenLDAP, WireGuard VPN, ClamAV et phpLDAPadmin en conteneurs Docker.

⏱️ 23 jours – Technologies : Ubuntu, Docker, Vagrant, OpenLDAP, WireGuard, UFW
Conception, installation et sécurisation d'une infrastructure réseau — École IRIS Nice

Conception et déploiement d'une infrastructure réseau segmentée avec authentification 802.1X via FreeRADIUS + OpenLDAP sur matériel Cisco réel.

⏱️ 5 jours – Technologies : Cisco IOS, FreeRADIUS, OpenLDAP, Docker, Debian 12, WireGuard, Nextcloud, ClamAV
Conception et mise en place d'un PRA/PCRA avec automatisation des sauvegardes — Campus Mediaschool

Conception et déploiement d'un Plan de Reprise d'Activité complet avec BorgBackup chiffré AES-256, supervision Prometheus/Grafana et test de restauration validé sur maquette Vagrant.

⏱️ 20 jours — Technologies : BorgBackup, Borgmatic, Prometheus, Grafana, Vagrant, Ubuntu
