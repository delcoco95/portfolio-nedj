Conditions de réalisation1 (ressources fournies, résultats attendus) 
 
Ressources : Un cahier des charges simple précisant l’objectif : automatiser la configuration de rôles Windows 
Server 2022 via un menu PowerShell. Un environnement de test en machine virtuelle (VM Windows Server 
2022). 
 
Résultats attendus : Script PowerShell  avec menu interactif qui permet le déploiement et configuration des 
services réseau requis (par exemple, Active Directory, DHCP, DNS) sur Windows Serveur. Documentation 
détaillée des procédures d’installation et de configuration.

Réseau local (NAT).

[10] Import CSV. 
[11] Quitter.
